Documentation

Introduction

This documentation will help you get familiar with the resources of the Formula 1 API and show you how to make different queries, so that you can get the most out of it.

REST

Base url: http://localhost:5039/api/

The base url contains information about all available API's resources. All requests are GET requests and go over https. All responses will return data in json.

There are three available resources:

  • Drivers : used to get all drivers.
  • Teams : used to get all teams.
  • Races : used to get all races.

GET

http://localhost:5039/api/

"drivers" : http://localhost:5039/api/drivers

"teams" : http://localhost:5039/api/teams

"races" : http://localhost:5039/api/races

Info

  • The drivers resource contains the name, age, and nationality of each driver. In addition, the resource contains the name of the image file of the specific driver and its unique ID.
  • The teams resource contains the manufacturer (for example Ferrari), driver1 and driver2 of each team. In addition, the resource contains the name of the image file of the specific team and its unique ID.
  • The races resource contains the name of the winner, the winner time, the number of laps in the specific race and the GrandPrix (country where the race took place). In addition, the resource contains the unique ID of each race.

Accessing drivers

Get all drivers:

http://localhost:5039/api/drivers

Get driver by id:

http://localhost:5039/api/drivers/{id of driver}

Get driver by name:

http://localhost:5039/api/drivers/ GetDriverByName/{name of driver}

Accessing teams

Get all teams:

http://localhost:5039/api/teams

Get team by id:

http://localhost:5039/api/teams/{id of team}

Get team by manufacturer:

http://localhost:5039/api/teams/ GetTeamByManufacturer/ {name of manufacturer}

Accessing races

Get all races:

http://localhost:5039/api/teams

Get race by id:

http://localhost:5039/api/races/{id of race}

Get race by grandPrix:

http://localhost:5039/api/races/ GetRaceByGrandPrix{name of GrandPrix}
